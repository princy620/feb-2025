# 🤖 Gemini SQL Assistant

An intelligent, LLM-powered application that transforms natural language into SQL queries, executes them on a database, and explains the SQL logic — all within a clean and interactive Streamlit interface.

## 🔍 Project Overview

This app enables users to:
- ✅ Ask questions in **plain English**
- ✅ Generate **SQL queries** using **Gemini 2.0 Flash**
- ✅ Run them on a **live SQL Server or SQLite** database
- ✅ View results in a structured format
- ✅ Understand the logic with **LLM-generated step-by-step SQL explanations**

💡 Designed to make structured data querying more **intuitive**, **interactive**, and **educational**.

---

## 🛠️ Tech Stack

- 🔹 **Google Gemini 2.0 Flash** (`google-generativeai`)
- 🔹 **Streamlit** for building the UI
- 🔹 **SQLite** or **Microsoft SQL Server**
- 🔹 **Python** (`pyodbc`, `sqlite3`, `pandas`)
- 🔹 `.env` for secure API key handling

---

## 🧠 Features

- ✨ **Natural Language to SQL** conversion using Gemini
- 📊 **Live SQL Execution** on your database
- 📘 **LLM-powered Explanation** of SQL logic
- 🧰 Sample prompts to guide user interaction
- ⚙️ Modular functions for prompt engineering and query execution
